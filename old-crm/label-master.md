# Label Master — CRM Google Contacts

**Rule:** never invent labels outside this list mid-capture. Add new labels here first.

---

## Tier labels (apply exactly one per contact)

| Label     | Who                                              | Cadence       | Target size |
|-----------|--------------------------------------------------|---------------|-------------|
| `CRM-T1`  | Live deal or active conversation                 | Weekly        | 10–30       |
| `CRM-T2`  | Referrers, repeat contacts, mentors              | Monthly       | 30–60       |
| `CRM-T3`  | Actively want to stay connected; future optionality | Quarterly  | 50–150      |
| *(none)*  | T4 Archive — searchable, no outreach             | Never         | Rest        |

**Tier decision rule:** "If I lose touch with this person for a year, does it hurt the business?"
- Yes, clearly → T1 or T2
- Maybe, context-dependent → T3
- No → T4 (no label)

**Batch capture default:** always T4 (no label).

## Lifecycle labels (apply / update over time)

| Label          | When to apply                                         |
|----------------|-------------------------------------------------------|
| `Met-2026`     | Set once at capture; year you first met them          |
| `Touched-2026` | Add when you proactively reach out in that year; remove and update each January |

---

## Rules for adding new labels

1. Add it here first, with a description.
2. Use the format conventions above (no spaces, consistent casing).
3. If a new label duplicates an existing one, use the existing one.
4. Review label inflation quarterly — if the list grows beyond 40 entries, prune.

---

## Google Contacts: how to create labels

1. Open [Google Contacts](https://contacts.google.com) on desktop.
2. Left sidebar → **+ Create label**.
3. Create each label in the Tier and Lifecycle sections now (before you start capturing).
4. Create Event and Theme labels as you encounter them.

---

## Searching in Google Contacts

- By label: type `label:CRM-T1` in the search bar.
- By event: type `label:Event-Milken-2026`.
- By theme: type `label:Theme-LogisticsAsia`.
- By name/company: just type the name.
